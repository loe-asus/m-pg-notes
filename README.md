# m-pg-notes

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面，主要存放一些零散的页面笔记、小型演示页面或临时页面。不针对任何特定域名或网站，所有页面均为独立内容。

## 目录说明

```
m-pg-notes/
├── docs/                # 页面归档主目录
│   ├── page-01/        # 单个页面文件夹（示例）
│   ├── page-02/
│   └── ...
├── assets/              # 公共资源（如有）
└── README.md            # 本文件
```

- `docs/` 目录下按页面分别存放，每个页面为一个独立文件夹，包含对应的 HTML 文件及相关资源。
- 页面命名以 `page-` 开头后接序号，便于管理和扩展。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可直接在浏览器中打开查看。
- 页面内容可能包含笔记、测试、演示或临时记录，不构成任何正式产品。
- 部分页面可能引用外部资源（如 CSS 库、字体等），请确保网络连接正常。
- 页面之间无关联，可单独使用。

## 维护说明

- 本仓库不定期更新，新增页面或修改现有页面。
- 若需提交页面，请创建新文件夹至 `docs/` 目录，并确保页面名称不重复。
- 不建议直接修改他人页面，如有需要可提交 Issue 或 Pull Request。
- 仓库维护者保留对页面内容的最终处理权。

## 其他

- 本仓库仅用于归档和展示，不提供任何形式的服务或接口。
- 如有疑问或建议，欢迎通过 GitHub Issues 联系。
